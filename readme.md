# RentIt

---

This is a Management Booking system made with Laravel 5.1.

## Installation
---

Prerequisite : node , composer , lamp

steps:
---

1. run `php artisan migrate`

2. run `php artisan db:seed`

3. run `php artisan serve`

To get more info visit laravel docs.
